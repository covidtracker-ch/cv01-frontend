# Covid-Tracker Frontend

## Project Setup
`gem install middleman`

## Local Development
`middleman serve`

## Production Build
`middleman build`

## Deployment
Deployment is automated on Netlify.
The development-branch deploys here: https://staging.covidtracker.ch
